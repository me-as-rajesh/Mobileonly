# **App Name**: ConnectCell

## Core Features:

- User Authentication & Profiles: Users can register, log in using Firebase, manage their profiles, and view other users' public profiles.
- Product Listing & Management: Sellers can create new mobile phone listings with multi-step forms, upload images to Cloudinary, specify device details, and manage their active or sold listings.
- Advanced Listing Search & Filtering: Buyers can browse a grid of listings, apply advanced filters based on brand, model, price, condition, and location radius.
- Real-time Buyer-Seller Chat: Users can engage in real-time, anonymous chat conversations about listings via Firebase Realtime Database, with optional contact sharing by the seller.
- AI Price Suggestion Tool: When creating a listing, sellers receive an AI-generated suggested resale price range (tool) based on device specifications (e.g., brand, model, condition, purchase year).
- AI Listing Moderation Tool: New listings are automatically checked by an AI tool for suspicious content, potential fraud, or spam, with flags for admin review.
- Favorite & Recently Viewed Listings: Buyers can save listings to their favorites list and easily access their recently viewed items.

## Style Guidelines:

- Light color scheme to keep focus on product images and create an open, modern feel.
- Primary brand color: `#5477B7`. A sophisticated, calm blue that conveys trustworthiness and technological essence.
- Background color: `#ECF0F7`. A very light blue-grey that softly complements the primary color without being distracting.
- Accent color: `#28A9BF`. A vibrant and refreshing aqua blue, chosen for its strong contrast and modern energy, complementing the primary blue-grey palette effectively.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly techy, and clean appearance in titles.
- Body font: 'Inter' (sans-serif) for high readability and a neutral, professional look suitable for listing descriptions and general text.
- Utilize simple, clean, and modern vector-based icons that intuitively represent marketplace actions like favoriting, chatting, or listing management.
- Implement a responsive, grid-based layout for browsing listings, ensuring an intuitive user experience across various device sizes. Prioritize clear visual hierarchy for product information.
- Subtle and functional animations for UI transitions, loading states, and user interactions to enhance feedback without being overly distracting.
